3. What is Event Bubbling? And how does it work?

Ans: Event Bubbling is a process where an event starts from the target element and then propagates upward through it's parent elements up to document.

4. What is Event Delegation in JavaScript? Why is it useful?

Ans: Event Delegation is a technique where you attach one event listener to a parent element instead of multiple listeners to its child elements, and handle events using event bubbling.

5. What is the difference between preventDefault() and stopPropagation() methods?

 Ans:
 preventDefault() - stops the browser’s default action.

stopPropagation() - stops the event from bubbling up to parent elements.
